# ctrdata

The `ctrdata` R package provides tools to access and extract clinical trial data from DSiNÉ Cameroon and its affiliated platforms. It supports authenticated data access with an API token, as well as anonymous access to public datasets.

---

## 📦 Installation

To install the package from GitHub:

```r
# Install devtools if needed
install.packages("devtools")

# Install the ctrdata package
devtools::install_github("dsinecameroon/ctrdata")
